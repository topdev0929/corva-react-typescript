# Main Utils

Common utils.

## closest

Copies text to clipboard

### Example

```javascript
import { main } from '~/utils';

main.copyToClipboard('some text to copy'); // returns undefined, text is copied to clipboard
```

<div class="param">
  <div class="code-block small">
    <i>text: String</i>
  </div>
  <div class="chip">Required</div>
</div>
<div class="param-description">Text to copy.</div>
