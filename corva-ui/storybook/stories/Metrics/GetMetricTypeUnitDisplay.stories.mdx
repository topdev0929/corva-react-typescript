# Convert Utils

Util for work with metric values.

## getMetricTypeUnitDisplay

Returns getMetricUnitDisplay by provided metric type.

### Example

```javascript
import { getMetricTypeUnitDisplay } from '~/utils/metrics';

getMetricTypeUnitDisplay('build_rate');
```

<div class="param">
  <div class="code-block small">
    <i>metricType: String</i>
  </div>
  <div class="chip">Required</div>
</div>
