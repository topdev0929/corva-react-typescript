# Main Utils

Common utils.

## intToRGB

Transform given number to a color (RGB). Returns String.

### Example

```javascript
import utils from '~/utils';

utils.intToRGB(42); // returns '00002A';
```

#### Parameters

<div class="param">
  <div class="code-block small">
    <i>i: Number</i>
  </div>
  <div class="chip">Required</div>
</div>
