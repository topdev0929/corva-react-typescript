# Main Utils

Common utils.

## getLinesFrequency

Get lines friquency to get from CSV. Helps to get from 1000 to 9999 lines from CSV to preven complex calculations.

### Example

```javascript
import utils from '~/utils';

utils.getLinesFrequency(42); // returns 42
utils.getLinesFrequency(9999); // returns 1
```

#### Parameters

<div class="param">
  <div class="code-block small">
    <i>linesAmount: Number</i>
  </div>
  <div class="chip">Required</div>
</div>
