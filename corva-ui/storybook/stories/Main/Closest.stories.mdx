# Main Utils

Common utils.

## closest

Finds closest to value in array of numbers.
Source: https://stackoverflow.com/a/19277804/8337033

### Example

```javascript
import utils from '~/utils';

utils.closest([1, 2, 42], 33); // returns 42
```
