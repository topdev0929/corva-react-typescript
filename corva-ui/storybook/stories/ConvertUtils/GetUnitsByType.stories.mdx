# Convert Utils

Convert Utils is a tool for metric data convertioning.

## getUnitsByType

Lists all the units of a specific type. Returns Array of Strings.

### Example

```javascript
import { getUnitsByType } from '~/utils/convert';
getUnitsByType('length'); // returns ['mm', 'cm', 'm', 'km'] or ['in', 'yd', 'ft', 'mi'], depends on current unit system.
```
