# Main Utils

Common utils.

## parseNumberFromString

Parse number from string, if not number - returns 0.

### Example

```javascript
import utils from '~/utils';

utils.parseNumberFromString('someCSV'); // returns 0;
utils.parseNumberFromString('42'); // returns 42;
```

#### Parameters

<div class="param">
  <div class="code-block small">
    <i>string: String</i>
  </div>
  <div class="chip">Required</div>
</div>
