# Main Utils

Common utils.

## getOperationByType

Gets operation by operation type. Returns object or undefined.

### Example

```javascript
import utils from '~/utils';

utils.getOperationByType(operationType);
```

#### Parameters

<div class="param">
  <div class="code-block small">
    <i>operationType: Number|null</i>
  </div>
  <div class="chip">Required</div>
</div>
