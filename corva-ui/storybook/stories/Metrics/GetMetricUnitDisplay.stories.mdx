# Convert Utils

Util for work with metric values.

## getMetricUnitDisplay

Returns formatted unit to display according to provided metric config.

### Example

```javascript
import { getMetricUnitDisplay } from '~/utils/metrics';

const metricConfig = {
  formatUnitDisplay,
  to,
  unitType,
}; // some metrics config here

getMetricUnitDisplay(metricConfig); // if to and unitType are not provided â€“ returns ''
```
