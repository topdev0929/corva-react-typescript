# Convert Utils

Convert Utils is a tool for metric data convertioning.

## updateUserUnits

Updates current custom unit system with user data.

### Example

```javascript
import { updateUserUnits } from '~/utils/convert';
updateUserUnits({
  unit_system: {
    oilFlowRate: 'bbl/min',
    massFlowRate: 'lb/min',
    shortLength: 'in',
    msePressure: 'ksi',
  },
  company: {
    unit_system: {
      oilFlowRate: 'm3/min',
      massFlowRate: 'kg/min',
      shortLength: 'mm',
    },
  },
});
```
