# Convert Utils

Convert Utils is a tool for metric data convertioning.

## convertArray

Converts a property in a simple array of js objects or arrays.

### Example

```javascript
import { convertArray } from '~/utils/convert';
convertArray([{ value: 100 }, { value: 200 }], 'value', 'length', 'ft'); // returns [{value: 30.48}, {value: 60.96}] (m)
```
