# typography

@corva/ui library provides CSS classes that can be applied to any
element to make it look according to the Corva design system

```javascript
import { typography } from '@corva/ui/styles';

const Demo = () => <span className={typography.bold24}>Default</span>;
```

If you need to use an alternative text color from Corva design system
you can add its class, like this

```javascript
import { typography } from '@corva/ui/styles';
import classNames from 'classnames';

const Demo = () => (
  <span className={classNames(typography.bold24, typography.colors.t6)}>Alternative color</span>
);
```
