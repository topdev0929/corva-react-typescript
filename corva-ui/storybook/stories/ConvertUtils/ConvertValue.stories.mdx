# Convert Utils

Convert Utils is a tool for metric data convertioning.

## convertValue

Converts a single value. Returns number â€“ converted value.

### Example

```javascript
import { convertValue } from '~/utils/convert';
convertValue(100, 'length', 'ft'); // returns 30.48 (m)
```
