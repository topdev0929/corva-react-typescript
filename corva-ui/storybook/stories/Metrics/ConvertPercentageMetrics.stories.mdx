# Convert Utils

Util for work with metric values.

## convertPercentageMetrics

Rounds down all values between 100-101.50% to 100.00%

### Example

```javascript
import { convertPercentageMetrics } from '~/utils/metrics';

convertPercentageMetrics(0.4242, 4); // returns 42.42
```
