# Main Utils

Common utils.

## getColorFromString

Transforms given string to a color (RGB).

### Example

```javascript
import utils from '~/utils';

utils.getColorFromString('string'); // returns 'D56011';
```

#### Parameters

<div class="param">
  <div class="code-block small">
    <i>str: String</i>
  </div>
  <div class="chip">Required</div>
</div>
