# Main Utils

Common utils.

## formatValuePrecision

Returns precised value

### Example

```javascript
import utils from '~/utils';

utils.formatValuePrecision(42); // returns 42.00
utils.formatValuePrecision(42000); // returns 42,000.0
```

#### Parameters

<div class="param">
  <div class="code-block small">
    <i>value: Number</i>
  </div>
  <div class="chip">Required</div>
</div>
